!!Always pull most recent version from git before coding to avoid errors

1. Download nodejs: https://nodejs.org/en/download
2. npm install
3. frontend .gitignore:
    # Node modules
    node_modules/

    # Mac system files
    .DS_Store

    # VS Code settings
    .vscode/

    # Log files
    npm-debug.log*
4. run (make sure not in frontend but main file):
    node server.js
